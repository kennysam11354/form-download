인보이스 발행부터 미수금 관리까지 - 상세 운영 가이드

1. 인보이스 작성 및 발행

  • 대상 선택: `Invoices` 메뉴에서 `Create New Invoice` 클릭
  • 고객사 지정: 시스템이 해당 고객의 '미납 티켓' 목록 자동 검색
  • 티켓 검토: 인보이스에 포함할 작업을 체크하고 불필요한 건 제외
  • 발행: `Issue Invoice` 버튼 클릭
⚠️ 주의: 발행된 인보이스는 수정이 불가능하며, 수정 시 Void(무효화) 후 재발행해야 합니다.

2. 현장 사진 및 관리

  • 현장 촬영: 기사가 앱에서 작업 전/후 사진을 직접 촬영
  • 티켓 첨부: 모든 사진은 해당 서비스 티켓에 클라우드 방식으로 영구 보존
  • 청구서 포함: `Include Service Photos` 옵션 선택 시 인보이스에 증빙 자료로 자동 삽입

3. 미수금(AR) 조회 및 관리

  • 현황 파악: 대시보드의 'AR Aging Summary'에서 실시간 미납액 확인
  • 입금 등록: 대금을 수령하면 `Post Payment`를 통해 원장 반영
  • 금액 배정: 입금액을 특정 인보이스에 매칭하여 '완납(PAID)' 상태 관리

4. 서브 컨트랙터 및 고객 구좌 관리

  • 구좌 배정: 새 고객 발생 시 `Assign to Sub-contractor` 기능을 통해 담당 테크니션에게 연결
  • 구좌 이동: 필요 시 고객 구좌를 서브 컨트랙터 간에 상호 이관 (히스토리 유지)
  • 독립적 관리: 각 서브 컨트랙터는 본인에게 배정된 리스트만 관리 권한 가짐

5. 지능형 모니터링 및 자동 스테이트먼트

  • 필터링: 대시보드 상태를 서브 컨트랙터 그룹군별, 연체별, 매출액 순으로 필터링
  • Auto Statement: 발행된 티켓을 취합하여 스테이트먼트 자동 산출
  • 자동 발송: 설정된 특정 일자에 그룹별로 구분하여 고객에게 자동 이메일 발송
